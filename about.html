features

Available Offline

Allume uses HTML5's application cache to make it available offline. Just visit allume.cc once, and you are ready to go! On node.js, just install allume from the npm repository.

Custom Repositories

You can specify custom public repositories. At this point, we are investigating how to implement private, secure repositories. Help welcome!

Cross-Runtime

Allume is designed to run inside browsers and node.js. It should work on any browser that supports HTML5 features or any runtime that supports the node.js api.
